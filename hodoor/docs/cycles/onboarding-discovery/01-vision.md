# Vision — onboarding-discovery

## Scope mode
SELECTIVE EXPANSION

## 11-star version
Hodoor turns a casual walk through your home into a complete appliance inventory with zero friction. Every photo instantly identifies the appliance, resolves the exact model via QR/label when accessible (never asking the user to move furniture), estimates age and warranty window, and rewards each scan with an immediate, actionable maintenance tip. By the end of the tour, the user has a full household map, knows their top maintenance priority, and feels like they just hired a home care expert on WhatsApp.

## Commitment
Build a guided, conversational onboarding flow where Hodoor walks the user room-by-room through photo-scanning each appliance, confirms detections, captures acquisition dates, delivers a quick maintenance tip per appliance, flags obvious missing items, and closes with a prioritized maintenance recap.

## Not building
Push notification/reminder engine, recurring maintenance scheduler, multi-user household support, room/zone data model, appliance replacement or marketplace recommendations, warranty claim automation, integration with manufacturer APIs

## Why this scope
The core photo-scan and equipment-creation capabilities already exist. The gap is orchestration: turning isolated tool calls into a coherent, low-friction first experience that delivers value at every step. We expand selectively on the "first value moment" (instant maintenance tip per appliance) and completeness check (missing appliance detection), because these are what transform a data-entry chore into something that feels genuinely useful from minute one.
